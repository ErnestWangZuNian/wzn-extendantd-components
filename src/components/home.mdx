## 开发背景

  1. 尽管antd已经很强大了，但是最近在公司项目中发现仍然有些常用组件需要但antd不支持如NumberInput,MoneyInput,MobileInput等组件 
  2. 另外自己也想学习一下如何构建一个组件库，因此这个项目诞生了

##  安装使用


  `npm install wzn-extendantd-components`

##  项目收获

  学会了如何打包组件库,两种打包方式


     babale打包成lib es 模块
     webpack打包成 umd模块

  学会了如何给组件写文档 我采用的是 [docz](https://www.docz.site/) 感觉还是蛮简单的 大写的👍
  
  更加熟悉如何发布自己的npm包  

  了解到了整个组件库的开发流程 也感受到了大佬的辛苦 特别是像antd组件库的创作者 和贡献者  欢迎有志同道合的小伙伴一起交流
